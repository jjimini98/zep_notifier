# Zep Environment Private Message Notifier

ZEP 공간에서 Private 채팅 메시지가 도착하면 OS 알림으로 알려주는 Chrome 확장 프로그램입니다.

## ✨ 주요 기능

- ZEP의 **Private 채팅 탭에서 새 메시지가 도착하면 즉시 알림**
- 접속 직후 기존 채팅 기록은 알림하지 않음
- 내가 보낸 메시지는 자동으로 제외
- 알림 클릭 시 ZEP 탭으로 이동
- 과도한 알림 방지를 위한 중복 및 쿨다운 처리

## 🔒 동작 방식

이 확장 프로그램은 ZEP 페이지 내부의 채팅 DOM 변화를 감지하여
새로운 메시지가 추가되었을 때만 알림을 생성합니다.

메시지 데이터는 외부 서버로 전송되지 않으며,
모든 처리는 사용자의 브라우저 내에서만 이루어집니다.

## 📌 필요한 권한

- **notifications**  
  → 새 메시지를 OS 알림으로 표시하기 위해 필요합니다.

- **storage**  
  → 사용자 닉네임을 로컬에 저장하여 본인 메시지를 제외하기 위해 필요합니다.

- **tabs**  
  → 알림 클릭 시 ZEP 탭으로 이동하기 위해 필요합니다.

- **zep.us 접근 권한**  
  → ZEP 채팅 메시지를 감지하기 위해 필요합니다.

## ⚠️ 주의 사항

- 이 확장 프로그램은 ZEP의 UI 구조 변경 시 동작이 영향을 받을 수 있습니다.
- Private 탭이 활성화된 상태에서만 알림이 작동합니다.
