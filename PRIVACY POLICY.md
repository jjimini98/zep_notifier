# Privacy Policy

Effective Date: 2026-02-27

ZEP Private Chat Notifier does not collect, store, or transmit personal data to any external server.

## Data Collection

This extension does NOT:
- Collect personal information
- Transmit chat messages to external servers
- Track browsing activity outside of zep.us

## Data Storage

The extension may store the following data locally in the user's browser:

- User nickname (to exclude self-sent messages from notifications)
- Internal settings related to notification behavior

All stored data remains in the user's local browser storage and is never transmitted externally.

## Permissions Usage

- notifications: Used to display OS-level notifications for new messages.
- storage: Used to store user nickname locally.
- tabs: Used to focus the ZEP tab when a notification is clicked.
- host permission (https://zep.us/*): Used to detect new chat messages within ZEP.

## Third-Party Services

This extension does not use analytics, tracking scripts, or third-party services.
